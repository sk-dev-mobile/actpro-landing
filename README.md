# АктПро — лендинг

Статическая одностраничная маркетинговая витрина продукта АктПро.

## Локальный просмотр

Файлы полностью статические, сборка не требуется:

```bash
python3 -m http.server 8000
```

Затем открыть `http://localhost:8000/`.

## Структура

- `index.html` — вся разметка страницы
- `styles.css` — стили
- `script.js` — мобильное меню (единственная интерактивность)
- `assets/` — логотип и favicon, скопированы из `actpro-product` (`apps/frontend/public/`)

## Ссылки на прод

Ссылки на регистрацию/вход захардкожены на `app.lgtime.ru` в `index.html`
(секция hero и шапка) — при переезде на другой домен поменять там же.
